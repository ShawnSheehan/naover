## Development

1 - install dependencies
`yarn || npm install`

2 - run development server `localhost:8080`
`yarn start|| npm start`

3 - lint via eslint
`yarn run lint || npm run lint`

4 - build application
`yarn run build:prod || npm run build:prod`
